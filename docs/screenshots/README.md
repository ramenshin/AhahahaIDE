- **개인정보 노출 금지**: 절대 경로(`D:\Projects\...`), 다른 프로젝트명, Claude 계정명, 이메일, API 키 등 캡처 전 확인

- **파일명 규칙**: `kebab-case.png` (예: `main-view.png`, `quiksearch.png`)
